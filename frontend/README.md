# Pitch UI (Frontend Demo)

> [!CAUTION]
> **This frontend is NOT evaluated by the Hackathon Grading Script.**
> The grading harness is completely headless. However, we are building this UI to generate a **Pitch Video** to maximize our aesthetic points.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Your Master Guide

For detailed tasks, design guidelines, and the plan for integrating with the local backend to record the demo, you MUST read the main frontend documentation:

👉 **[docs/demo/PITCH_FRONTEND.md](../docs/demo/PITCH_FRONTEND.md)**

## Architecture
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS + Vanilla CSS (`globals.css`)
- **Animations:** Framer Motion
- **Icons:** `lucide-react`, `reicon-react`
