# Role: Member 1 - Frontend (MERN/Next.js)

## Focus Areas
You are responsible for the physical user interface of the application. The judges will evaluate the product based on its visual excellence and premium feel.

## Required Skills / Best Practices
- **Next.js App Router**: Build server-side and client-side components correctly.
- **TypeScript**: Strictly type your props, API responses, and states.
- **Design Aesthetic**: Use Vanilla CSS (no generic Tailwind components). Implement Stitch-inspired aesthetics: glassmorphism, fluid micro-animations, curated color palettes, and modern typography (e.g., Inter or Outfit). Do NOT output "AI slop".

## Initial Tasks (Phase 1 & 2)
1. Clone the repo and navigate to `/frontend`.
2. Build the main routing dashboard UI shell.
3. Integrate the layout with the backend API (`/api/route`).
4. Ensure the UI visually demonstrates the "Routing Path" (showing when it routes to Semantic Layer vs XGBoost vs LLM) with clean animations.
